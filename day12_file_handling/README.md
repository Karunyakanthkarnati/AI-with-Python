# Day 12 – File Handling for AI Systems (Read & Write Text Files)

This project introduces **file handling in Python**, a core skill required
for building real-world AI, GenAI, and Agentic AI systems.

Instead of taking manual input every time, AI systems read and write data
from files such as:

- Documents
- Logs
- Chat history
- Resumes
- Knowledge base files (for RAG systems)

Day 12 focuses on safely reading and writing text files using Python.

---

## 📌 Topics Covered

- What is file handling?
- Opening files in different modes (`r`, `w`, `a`)
- Reading entire file content
- Writing data into files
- Appending content safely
- Why closing files is important
- Using `with open()` (industry-standard method)

---

## 🧠 Why File Handling Matters in AI

AI systems depend heavily on stored data.

File handling is used in:

- Retrieval-Augmented Generation (RAG)
- Chatbot memory storage
- Resume analyzers
- Log file analysis
- Dataset preprocessing
- Prompt storage and retrieval

Without file handling, you cannot build production-level AI systems.

## ⚠️ Important Concepts Learned

- `"w"` mode overwrites file content
- `"a"` mode appends without deleting
- `"r"` mode reads existing files
- `with open()` automatically closes files
- File paths must be correct to avoid `FileNotFoundError`
